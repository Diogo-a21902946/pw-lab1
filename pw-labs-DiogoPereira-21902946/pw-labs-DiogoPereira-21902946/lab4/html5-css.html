CSS

Cascading Style Sheets (CSS) é um mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web

Seletores CSS
Universal O seletor universal aplica o estilo a todos os elementos da página. Utilizei em todas as páginas.
Tipo O seletor de tipo usa o nome do marcador a estilizar. Utilizei em todas as páginas.
Classe O seletor de classe associa um nome a um estilo. Utilizei em todas as páginas.
Id O atributo id permite identificar univocamente um elemento do HTML. Utilizei na página HTML5-CSS.
Atributo O seletor atributo é a plicadoo a atributos de elementos. Utilizei em todas as páginas.
Pseudo-classe Seletor associado ao estado de um elemento. Utilizei em todas as páginas.
Pseudo-elemento O seletor pseudo-elemento aplica certas condições a partes de elementos. Utilizei em todas as páginas.

Combinadores de seletores CSS
Descendentes O combinador de seletores descendente corresponde a todos os elementos que são descendentes de um elemento especificado. Utilizei em todas páginas.
Filhos O combinador de seletores filho seleciona todos os elementos que são os filhos imediatos de um elemento especificado. Não utilizei.
Adjacentes O combinador de seletores irmão adjacente seleciona todos os elementos que são os irmãos adjacentes de um elemento especificado. Utilizei na página Introdução.
Agrupados O combinador de seletores agrupados agrupa os seletores. Não utilizei.
Imediatamente adjacentes O seletor de irmão em geral seleciona todos os elementos que são irmãos de um elemento especificado. Utilizei em todas as páginas.
